Creating a Missive API Key

Step 1

From Missive, open the Settings menu by clicking your profile icon.

Step 2

From the API category, under the Token subcategory, click New token.

Step 3

Choose a name for your token and enter it into the text box.

Step 4

You will now see the API key you have created. This will not be shown again, so make a record of it and store it somewhere safe. Treat this token as a password - anyone who has your token can potentially execute functions in Missive using your profile.
